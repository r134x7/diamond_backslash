# diamond\black mirroring keyboard layout kbd file

The kanata.kbd file is being used as a proof of concept to demonstrate
how the diamond\black mirroring keyboard layout works.

More details including bugs are in the kanata.kbd file.

The kanata.kbd file is used with Kanata, see Kanata repo source for how Kanata works: https://github.com/jtroo/kanata
